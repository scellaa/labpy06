penjelasan code

1. data_mahasiswa_UPB = {}
   Dictionary untuk menyimpan semua data mahasiswa.
   
2. lihat_data()
   Menampilkan tabel data mahasiswa. Jika kosong → tampilkan “TIDAK ADA DATA”.

3. tambah_data()
   Input nama, NIM, nilai tugas/UTS/UAS, lalu hitung nilai akhir.
   Simpan data ke dictionary. Cek duplikasi nama sebelum disimpan.

4. ubah_data()
   Input nama yang ingin diubah. Jika ada → input data baru.
   Hapus data lama lalu simpan data baru.

5. hapus_data()
   Hapus data mahasiswa berdasarkan nama. Jika nama tidak ada → tampilkan “Data invalid”.

6. cari_data()
   Cari dan tampilkan data mahasiswa berdasarkan nama. Jika tidak ditemukan → tampilkan “Data invalid”.

7. Loop Utama (while True)
   Menampilkan menu: Lihat, Tambah, Ubah, Hapus, Cari, Keluar. Pilihan huruf menentukan fungsi yang dijalankan.
   “k” → keluar dari program.
